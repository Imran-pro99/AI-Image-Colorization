# Pre-trained Model

The model files are **not included** in this repository to respect the work of the original authors.

## Required Files

Download the following files from the official project:

- colorization_deploy_v2.prototxt
- colorization_release_v2.caffemodel
- pts_in_hull.npy

Official Repository:

https://github.com/richzhang/colorization

After downloading, place the files inside this folder:

```text
Model/
├── colorization_deploy_v2.prototxt
├── colorization_release_v2.caffemodel
└── pts_in_hull.npy
```
